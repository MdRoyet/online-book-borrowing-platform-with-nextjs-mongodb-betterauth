# 📚 BookedBorrow - Digital Library Management System

A sleek, modern, and highly interactive digital library application built with **Next.js 15**. BookedBorrow allows users to explore a vast collection of books, manage their profiles, and experience a seamless borrowing workflow with a focus on high-end UI/UX.

---

## 🚀 Live Link

[**View Live Demo**](https://online-book-borrowing-platform-with.vercel.app/)

## 🎯 Purpose

The purpose of BookedBorrow is to provide a user-friendly platform for book enthusiasts to discover new reads and manage their digital library interactions. It bridges the gap between static book catalogs and dynamic user experiences by implementing real-time search, 3D interactive elements, and secure authentication.

---

## ✨ Key Features

- **🔐 Robust Authentication:** Powered by **Better Auth** with MongoDB integration, supporting both Email/Password and Google Social Login.
- **🔍 Advanced Search & Filtering:** Real-time search bar and a functional category sidebar to filter books by genres (Tech, Story, Science, etc.).
- **🖼️ 3D Spotlight Slider:** A visually stunning 3D Coverflow slider for featured books using **Swiper.js**.
- **👤 Dynamic User Profile:** A private profile route allowing users to view their details in a structured table and update their information (Name & Photo).
- **📱 Responsive & Private Routes:** Fully responsive design using **Tailwind CSS** and **DaisyUI**, with protected routes for book details and profile management.
- **🎭 High-End Animations:** Smooth entrance animations and interactive "404" effects using **Framer Motion**, **Animate.css**, and **React-Spring**.
- **🔔 Instant Feedback:** Integrated **React-Toastify** for real-time success/error notifications during auth and book interactions.

## 🛠️ Technology Stack & Dependencies

This project leverages the latest web technologies to ensure speed, security, and beauty:

### Core Framework

- **Next.js 15:** React framework for production.
- **MongoDB:** NoSQL database for user and session storage.

### UI & Styling

- **Tailwind CSS & DaisyUI:** For utility-first styling and pre-built modern components.
- **React Icons:** Comprehensive icon library.
- **React Fast Marquee:** For the smooth, infinite scrolling text on the homepage.

### Authentication

- **Better Auth:** A comprehensive auth solution for Next.js.
- **MongoDB Adapter:** Seamless connection between auth sessions and the database.

### Animation Libraries

- **Framer Motion:** For layout transitions and grid animations.
- **Swiper.js:** For the interactive 3D Coverflow book slider.
- **Animate.css:** For CSS-based entrance and attention-seeking animations.
- **React-Spring:** For physics-based floating animations on the 404 page.
